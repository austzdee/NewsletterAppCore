# NewsletterAppCore (ASP.NET Core MVC)

A secure **ASP.NET Core MVC** newsletter subscription app that demonstrates a full, real-world workflow: user sign-up, validation, database persistence, and scalable project structure using **Entity Framework Core**.

## Overview
NewsletterAppCore is a learning + portfolio project built to practice modern .NET web development with clean, maintainable patterns. 
It focuses on building a reliable subscription flow backed by SQL Server, with strong validation and a structured MVC approach.

## Features
- Newsletter subscription form (First name, Last name, Email)
- Server-side validation (ModelState + DataAnnotations)
- Database persistence with **EF Core (Code-First)**
- SQL Server / LocalDB integration
- Clean MVC separation (Controllers / Views / ViewModels / Data)
- Ready for extension: email confirmation, admin dashboard, unsubscribe, export list

## Tech Stack
- **C#**
- **ASP.NET Core MVC**
- **Entity Framework Core**
- **SQL Server / LocalDB**
- **LINQ**
- **Bootstrap (optional, if used in your UI)**
- **Git & GitHub**
